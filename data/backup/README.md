# Backup Data

Deze map bevat backup en migratie bestanden.

## Bestanden

- `bestellingen.csv.migrated` - Gemigreerde bestellingen data
- `klanten.csv` - Backup klanten data

## Opmerking

Deze bestanden zijn backups of migratie bestanden. Ze worden niet gebruikt door de applicatie, maar worden bewaard voor referentie of herstel.

# Logs

Deze map bevat log bestanden en tijdelijke output bestanden.

## Bestanden

- `app.log` - Applicatie log
- `app_errors.log` - Error log
- `bon.txt` - Tijdelijke bon output

## Opmerking

Deze bestanden worden automatisch gegenereerd door de applicatie. Ze kunnen veilig worden verwijderd of gearchiveerd.

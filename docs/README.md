# Documentatie Map

Deze map bevat alle documentatie en uitleg bestanden voor het Pizzeria Management Systeem.

## Structuur

```
docs/
├── README.md (dit bestand)
├── KLANTGEGEVENS_VERBETERINGEN.md
├── PERFORMANCE_OPTIMIZATIONS.md
├── PAYMENT_SYSTEMS_COMPARISON.md
├── QUICK_TEST_GUIDE.md
├── TEST_STRATEGY.md
├── PRD.md
├── SECURITY_AND_INTEGRATION_GUIDE.md
├── PYTHON_WEB_CAPABILITIES.md
├── WEB_DEVELOPMENT_GUIDE.md
├── TEST_CHECKLIST.md
├── TEST_RESULTS_SUMMARY.md
├── TESTING_GUIDE.md
├── GITHUB_SETUP.md
└── pizzeria-web/
    ├── backend/
    │   ├── *.md (backend documentatie)
    └── frontend/
        └── *.md (frontend documentatie)
```

## Belangrijke Documentatie

### Algemene Documentatie
- **PRD.md** - Product Requirements Document
- **TESTING_GUIDE.md** - Test handleiding
- **TEST_STRATEGY.md** - Test strategie
- **PERFORMANCE_OPTIMIZATIONS.md** - Performance optimalisaties

### Setup & Configuratie
- **GITHUB_SETUP.md** - GitHub setup instructies
- **SECURITY_AND_INTEGRATION_GUIDE.md** - Security en integratie gids

### Web Applicatie
- **pizzeria-web/** - Alle web applicatie documentatie
  - Setup instructies
  - Deployment guides
  - Troubleshooting guides
  - Feature overzichten

### Verbeteringen & Vergelijkingen
- **KLANTGEGEVENS_VERBETERINGEN.md** - Voorstellen voor klantgegevens verbeteringen
- **PAYMENT_SYSTEMS_COMPARISON.md** - Vergelijking betalingssystemen

## Opmerking

De belangrijkste README.md bestanden blijven op hun originele locatie:
- `/README.md` - Hoofd README
- `/pizzeria-web/README.md` - Web app README
- `/pizzeria-web/backend/README.md` - Backend README
- `/pizzeria-web/frontend/README.md` - Frontend README
